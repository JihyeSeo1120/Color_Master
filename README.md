# Color Master
#### 2019 웨일 브라우저 확장앱 개발 콘테스트
<br/>


> **『웹 사이트의 메인 색상을 추출』**
> - 방문한 웹 사이트의 메인 색상에 대한 RGB & HEX 색상 코드를 제공합니다.
>
> **『화면에 보이는 색 PICKER』**
> - PICKER를 통해 선택 이미지에 대해 RGB&HEX 색상 코드 뿐만 아니라 색상 조절창을 제공합니다.
